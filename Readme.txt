This Repo aims to take a crude look at venture investments by major exchanges by 
scraping the appropriate venture page. It's meant to serve as a documentation of how data was sourced


To run yourself:
1. Update the html to scrape following the ReadMe.txt in each folder
2. Run CexVentures.py
